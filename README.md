## Available commands

### Image generation

#### Hat

Adds a hat on top of a characters portrait>

```
?hat <name> <surname> <server> <hat>
```
- the hat parameter is optional, it accepts one of these options
  - propeller (default)
  - nerd
  - dunce
  - dimmadome

#### Compatibility

Checks the compatibility between two characters or a character and an object. (might not work rn oopsie)

```
?compatibility "<name> <surname> <server>" "<name> <surname> <server>"
```

```
?compatibility "<name> <surname> <server>" "<thing>" -o
```

### Quotes

#### makeQuote

Generates a quote image bypassing the usual number of reactions needed

```
?makeQuote @someone "the quote you want to generate"
```

### Admin stuff

#### Registration

Some of the features of the bot need your character to be registered (eg. the undertale sleeper agent)>
This commands registers a ffxiv character and links it to your discord account

```
?addme <name> <surname> <server>
```

### Random

#### In ponkers conversion

Transform a length in cm to a length in ponkers

```
?inponkers <some number>
```
